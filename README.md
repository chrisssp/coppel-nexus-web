## About Coppel Nexus Web

A feature-rich online storefront built with Next.js 15 and React 19, designed to provide a seamless shopping experience with fast product search, intuitive browsing, and a smooth checkout flow.

## Features

- Product catalog with search and filtering
- Shopping cart and checkout flow
- Server-side rendering for fast page loads
- Responsive design for all devices

## Quick Start

### Prerequisites

- Node.js 18+
- npm

### Setup

```bash
git clone https://github.com/chrisssp/coppel-nexus-web.git
cd coppel-nexus-web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
