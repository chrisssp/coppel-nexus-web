## About Coppel Nexus Web

A web admin panel built for Coppel's field acquisition teams during the Talent Land 2025 Hackathon (4th place). It enables tracking micro-business referrals to Coppel Emprende, monitoring course progress, and managing collaborator routes and workload.

Part of a larger ecosystem that includes a mobile app for field collaborators and an Express/MongoDB/Firebase backend.

### Ecosystem

| Component | Stack |
|-----------|-------|
| Web Admin | Next.js 15, React 19, Tailwind CSS 4, TypeScript |
| Mobile App (Field Collaborators) | React Native, Expo, Tailwind CSS |
| Backend API | Express, MongoDB, Firebase |

## Features

- Register and track micro-business referrals to Coppel Emprende
- Monitor course progress for referred businesses
- Manage collaborator routes and workload distribution
- Real-time tracking of referral pipeline

## Quick Start

### Prerequisites

- Node.js 18+
- npm

### Setup

```bash
git clone https://github.com/chrisssp/coppel-nexus-web.git
cd coppel-nexus-web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
